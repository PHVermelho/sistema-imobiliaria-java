# 🏢 Sistema Imobiliária - Java

Sistema desktop desenvolvido em Java com Swing e MySQL para gerenciamento de uma imobiliária.

## 🚀 Funcionalidades

- Cadastro de usuários (Admin / Corretor)
- Login com validação
- Integração com banco de dados MySQL
- CRUD completo de usuários

## 🛠 Tecnologias

- Java
- Swing
- MySQL
- JDBC

## 📂 Estrutura

- MVC (Model, View, Controller)
- DAO Pattern
